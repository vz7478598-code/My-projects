# Архитектура приложения «Финансовый чат-ассистент»

## Что это за приложение

Десктопное Windows-приложение в виде чата/мессенджера. Пользователь перетаскивает банковскую выписку (PDF или скриншот) в окно, а приложение «отвечает» финансовой аналитикой: баланс, категории расходов, графики трендов.

## Общая схема потока данных

```
Пользователь drag-and-drop файл
        │
        ▼
┌─────────────────────┐
│  Детектор типа файла │  ← python-magic / сигнатуры
│  (MIME + магич. байты)│
└────────┬────────────┘
         │
    ┌────┴────┐
    │         │
   PDF      Изображение (PNG/JPG)
    │         │
    ▼         ▼
┌────────┐  ┌──────────────────┐
│pdfplumber│  │ OpenCV/Pillow     │  ← предобработка
│(Этап 2) │  │ (grayscale,       │
└────┬───┘  │  бинаризация,     │
     │      │  шумоподавление,  │
     │      │  deskew)          │
     │      └────────┬─────────┘
     │               │
     │               ▼
     │      ┌─────────────────┐
     │      │  Tesseract OCR   │  ← или Cloud OCR
     │      │  (rus + eng)     │
     │      └────────┬────────┘
     │               │
     │               ▼
     │      ┌─────────────────┐
     │      │ Постпроцессор    │
     │      │ (очистка текста) │
     │      └────────┬────────┘
     │               │
     └───────┬───────┘
             │
             ▼  Извлечённый текст (str)
     ┌───────────────────┐
     │ Balance Finder     │  ← regex, ключевые фразы
     │ (Этап 2)          │
     └───────┬───────────┘
             │
             ▼  Баланс → сообщение в чат
     ┌───────────────────┐
     │ Transaction Parser │  ← regex, эвристики
     │ (Этап 3)          │
     └───────┬───────────┘
             │
             ▼  Список операций [{date, description, amount, type}, ...]
     ┌───────────────────┐
     │ Classifier         │  ← словарь ключевых слов
     │ (Этап 3)          │
     └───────┬───────────┘
             │
             ▼  Операции с категориями
     ┌───────────────────┐
     │ Aggregator         │  ← группировка, суммирование
     │ (Этап 3)          │
     └───────┬───────────┘
             │
             ▼  {категория: сумма}
     ┌───────────────────┐
     │ Matplotlib         │  ← круговая диаграмма (PNG)
     │ Chart Generator    │
     │ (Этап 3)          │
     └───────┬───────────┘
             │
             ▼  Диаграмма → в чат
     ┌───────────────────┐
     │ SQLite Storage     │  ← сохранение period_key, баланса,
     │ (Этап 4)          │     категорий
     └───────┬───────────┘
             │
             ▼  Исторические данные
     ┌───────────────────┐
     │ Trend Analyzer     │  ← сравнение T vs T-1
     │ + Line Chart       │  ← линейный график (Matplotlib)
     │ + Comment Generator│  ← текстовый комментарий
     │ (Этап 4)          │
     └───────┬───────────┘
             │
             ▼  График + комментарий → в чат
```

## Форматы данных между модулями

### Извлечённый текст (выход pdfplumber / Tesseract → вход Balance Finder)
```
Тип: str (многострочная строка)
Пример:
"Выписка по счёту 40817810000000000001\nЗа период с 01.03.2024 по 31.03.2024\n\n
Дата       Описание              Сумма\n01.03.2024 ПЯТЕРОЧКА             -1 250,00\n
02.03.2024 ЯНДЕКС.ТАКСИ          -350,00\n...\nИсходящий остаток: 75 000,50 руб."
```

### Найденный баланс (выход Balance Finder)
```python
{
    "amount": 75000.50,       # float, всегда в базовых единицах
    "currency": "RUB",        # str: "RUB" | "USD" | "EUR"
    "confidence": "high",     # str: "high" | "medium" | "low" (приоритет поиска)
    "raw_match": "Исходящий остаток: 75 000,50 руб."  # исходная строка
}
```

### Список операций (выход Transaction Parser)
```python
[
    {
        "date": "2024-03-01",          # str, формат YYYY-MM-DD
        "description": "ПЯТЕРОЧКА",    # str, оригинальное описание
        "amount": 1250.00,             # float, абсолютное значение
        "type": "expense",             # str: "expense" | "income"
        "raw_line": "01.03.2024 ПЯТЕРОЧКА -1 250,00"
    },
    ...
]
```

### Категоризированные данные (выход Classifier)
```python
[
    {
        "date": "2024-03-01",
        "description": "ПЯТЕРОЧКА",
        "amount": 1250.00,
        "type": "expense",
        "category": "Продукты"          # str, одна из предопределённых категорий
    },
    ...
]
```

### Агрегированные расходы (выход Aggregator → вход Matplotlib)
```python
{
    "Продукты": 15420.50,
    "Транспорт": 3050.00,
    "Кафе и рестораны": 5200.00,
    "ЖКХ и связь": 8500.00,
    "Здоровье": 1200.00,
    "Переводы": 10000.00,
    "Другое": 2150.00
}
```

### Запись в SQLite (одна строка в хранилище)
```python
{
    "period_key": "2024-03",           # str, YYYY-MM (уникальный ключ)
    "analysis_date": "2024-04-05T14:30:00",  # ISO datetime
    "file_name": "выписка_март.pdf",
    "total_balance": 75000.50,
    "total_income": 120000.00,
    "total_expense": 45520.50,
    "categories": {                    # JSON-строка в БД
        "Продукты": 15420.50,
        "Транспорт": 3050.00,
        ...
    }
}
```
